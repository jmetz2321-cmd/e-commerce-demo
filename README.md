# ShopVolt — E-Commerce Cart Prototype

A clickable single-page e-commerce cart prototype built with React, TypeScript, and Tailwind CSS.

## Features

- Product browsing with category filters (Electronics, Clothing, Accessories)
- Add to cart with toast notifications
- Cart view with quantity increment/decrement and item removal
- Order summary with subtotal, tax, and total
- Checkout modal with simulated payment form and validation
- View toggling between shop and bag

```bash

npm run dev
```

Open (https://jmetz2321-cmd.github.io/e-commerce-demo/?v=2) in your browser.
